# 0008. Styling architecture: historical record

## Status

Superseded by ADR 0012.

## Historical note

This ADR records the pre-UI-Core migration architecture. Its references to app-local tokens, Sprinkles, Ark primitives, and `src/components/ui` are not current implementation guidance. ADR 0012 defines the active Atomic CSS boundary; UI Core owns the Project Theme and reusable primitive APIs.
