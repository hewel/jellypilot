# Adopt JMSR Control Room design system

JMSR replaces Material Design 3 as the governing visual language with a desktop-first JMSR Control Room design system: dark-only, clean OLED surfaces with selective cinematic glass, indigo `#4f46e5` as the JMSR brand primary, and `#818cf8` as the Jellyfin/server secondary accent. This gives the app an owned media-control identity instead of a generic mobile Material feel, while keeping useful existing component class APIs stable during the cutover.
